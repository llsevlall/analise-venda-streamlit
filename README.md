# Aplicativo para análise de vendas

Projeto pessoal desenvolvido em Python com Streamlit para análise e visualização de dados de vendas.
Um aplicativo cujo objetivo é filtrar, adicionar, excluir e análisar dados de vendas de uma empresa fictícia.

### Funcionalidades 

- Filtros dinâmicos por loja, produto, vendedor e forma de pagamento
- Tabela interativa de vendas 
- Cadastro e exclusão de clientes 
- Preparado para expansão com dashboards analíticos 

<small>O aplicativo continua tendo algumas atualizações, principalmente na parte dos dashboards. As bibliotecas externas que 
utilizei foram: matplotlib, pandas e streamlit</small>
